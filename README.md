# MyProfessions - Profession viewer
MyProfessions is a World of Warcraft addon that gives you a unified overview of all your characters' professions across your account.


**Core features:**

• Tracks every character you log into, storing their professions, level, class, race, gender, gold, and rested XP in a saved database per faction

• Displays all characters in a sortable, scrollable table with columns for both primary professions, secondary skills (Fishing, Cooking, First Aid, Archaeology), level with rested XP indicator, class, and gold

• Color codes characters by faction (blue for Alliance, red for Horde) with race/gender portrait icons

• Shows rested XP percentage inline on the level column with a detailed tooltip showing amount, percentage, and estimated time to full rested

• Highlights your currently logged-in character in the table

• Right-click any character row to show, hide, or delete them from the database


**MyProfessions options:**

• Filter by faction, realm, and visibility

• Options panel to control toggle button visibility, reload button, faction display preference, and realm scope

• Database backup, cleanup, and restore functionality


**Technical:**

• Supports both Retail and MoP Classic with version-appropriate APIs

• Persists all settings and character data across sessions via SavedVariables
